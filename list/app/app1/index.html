APP 综合分析

MobSF & Mobexler 识别

MobSF 简介

移动安全框架（MobSF）是一款自动化的一体化移动应用程序安全分析框架，
 支持 Android / iOS / Windows 平台的渗透测试、恶意软件分析与安全评估。

它可执行静态和动态分析，支持多种文件格式（APK、XAPK、IPA、APPX），
 并提供 REST API 以便集成至 CI/CD 或 DevSecOps 流程中。

MobSF 的动态分析器可帮助进行运行时安全评估与交互式检测。

Frida & HOOK 实践

r0capture 特性：

  • 仅限 Android 平台（支持 Android 7~11）
  • 无视所有证书校验与绑定
  • 支持所有 TCP/IP 四层模型中的应用层协议（HTTP、WebSocket、FTP、SMTP、IMAP 等）
  • 通杀所有应用层框架（OkHttp、Retrofit、Volley 等）
  • 无视加固（整体壳、二代壳、VMP 均可）

Frida 简介：

Frida 是一款跨平台动态插桩工具，可在应用运行时注入代码， 监控与修改行为。支持 Java 层与 Native 层，兼容 Windows、Mac、Linux、Android、iOS。

环境准备

组件 说明
系统 Python3
模拟器 逍遥模拟器
工具 r0capture、frida-server、BurpSuite、Wireshark

Frida 下载地址：
🔗 https://github.com/frida/frida/releases

本地安装 Frida

pip install frida
pip install frida-tools

模拟器安装 Frida

注意：本地与模拟器的 Frida 版本必须一致

下载对应版本： https://github.com/frida/frida/releases

获取模拟器 CPU 架构

adb shell getprop ro.product.cpu.abi

上传并启动 frida-server

adb devices
adb push frida-server /data/local/tmp/
adb shell chmod 755 /data/local/tmp/frida-server
adb shell su /data/local/tmp/frida-server &

验证运行状态：

adb shell ps | grep frida

端口转发与连接验证

adb forward tcp:27042 tcp:27042

# 验证连接
frida-ps -U
frida-ps -R

Frida 绕过方案

方案一：Frida + r0capture + Wireshark

项目地址： 🔗 https://github.com/r0ysue/r0capture

# 模拟器执行
./frida-server

# 获取包名
ls /data/data

# 启动抓包
python r0capture.py -U -f com.example.app -p capture.pcap

方案二：Frida + Hook 脚本 + BurpSuite

项目地址： 🔗 https://github.com/apkunpacker/FridaScripts

# 模拟器执行
./frida-server

# 本地执行
frida -U -f com.example.app -l hook.js
